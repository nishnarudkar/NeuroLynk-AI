    Model Selection Insight

    Although SVM achieved the highest ROC AUC and KNN showed strong Macro F1 performance, XGBoost was selected as the final model.

    This is because XGBoost provides a strong balance between performance and interpretability. In medical applications, understanding model decisions is critical, and XGBoost allows extraction of meaningful speech biomarkers using SHAP explanations.

    Therefore, XGBoost is used for prediction and explainability in this system.
